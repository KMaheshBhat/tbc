# TBC Persist Memories Method Definition 0.1

This specification defines the Persist Memories method for the Third Brain Companion System.

## Specifications

- Method Name: Persist Memories
- Purpose: To record interactions and maintain memories at the end of every interaction.
- Actors: Agent (primary), Prime User (if manual)
- Steps:
  - To be used at the end of every interaction.
  - To be used using the `vault` system methods.
  - The Agent will use any tools available to record the interaction to its `vault` as specified in [Schema](#schema).
  - If the Agent is unable to record, the Agent can request the Prime User to do in its stead.
