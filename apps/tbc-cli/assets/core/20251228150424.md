# TBC Operational Fidelity (CLI) Specification 0.4

## Instantiate and Upgrade Companion (`init`)

* Allows a Prime User to instantiate a new Companion directory and upgrade an existing Companion directory.

```bash
$ tbc help init
Usage: tbc init [options]

Initialize a new TBC companion directory

Options:
  --upgrade           Allow re-initialization of existing companion
  --companion <name>  Name of the AI companion
  --prime <name>      Name of the prime user (group)
  -h, --help          display help for command
```

## 1. Environment Probing (`probe`)

* Identifies paths defined in the `root.md` YAML frontmatter and verifies access.

## 2. Integrity Validation (`validate`)

* Verifies that all files conform to their stated `record_type` schema and mandatory H2 section requirements.

## 3. Indexing (`dex`)

* Refreshes non-authoritative summary files in the `dex/` directory based on explicit paths in `root.md`.
