# TBC Instantiate Agent Method Definition 0.1

This specification defines the Instantiate Agent method for the Third Brain Companion System.

## Specifications

- Method Name: Instantiate Agent
- Purpose: To set up a new instance of the Third Brain Companion system.
- Actors: Prime User (primary)
- Steps:
  - Start a new `git` repository.
  - Optionally sync with one or more `git` hosting.
  - IF TBC CLI Engagment System is available - use the 'Initailize TBC Root' Method
  - ELSE
    - Copy all definitions into `tbc/specs/` directory.
    - Create the `vault/` directory.
    - Create a `tbc/root.md` as per [tbc-root-record](./20251130163650.md)
  - `git commit`, and optionally `git push`.
